# 图书商城系统 —— 产品说明文档

## 一、项目背景与目标
本项目为“图书商城系统”，是一个基于 Java Web 技术栈的教学与实践型 Web 应用。
系统模拟真实电商平台中的图书选购流程，面向普通用户提供图书浏览与购物车功能，
面向管理员提供图书、分类及用户的后台管理功能。

本项目严格按照软件工程规范进行设计与开发，采用分层架构与 Git 版本管理，
用于加深对 Servlet、JSP、JDBC、Session 等 Web 技术的理解和应用。

---

## 二、用户角色说明

### 1. 普通用户（USER）
普通用户可以通过系统完成以下操作：
- 用户注册与登录
- 浏览图书列表
- 按分类查看图书
- 查看图书详情
- 将图书加入购物车
- 查看与管理购物车内容

### 2. 管理员（ADMIN）
管理员由数据库中预先指定，不提供在线注册功能。管理员主要用于后台管理：
- 图书信息管理（新增、修改、删除、查询）
- 图书分类管理
- 用户信息管理（查询、删除）

---

## 三、系统功能概览

### 前台功能
- 用户注册
- 用户登录 / 退出
- 图书列表展示
- 图书分类浏览
- 图书详情查看
- 购物车管理（添加、查看、修改数量、删除）

### 后台功能
- 图书管理（CRUD）
- 分类管理（CRUD）
- 用户管理（查询、删除）

---

## 四、系统边界说明
本系统不涉及在线支付、订单管理、物流发货等复杂业务，
仅实现至“购物车”阶段，用于满足课程教学与实验要求。

---

## 五、技术选型
- 后端技术：Servlet、JSP、JDBC
- 会话管理：HttpSession
- 数据库：MySQL
- 前端技术：HTML、CSS（简洁风格）
- 项目管理：Git + GitHub
